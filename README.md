# **Instagram Clone**

## **Project Overview**

A full-stack Instagram clone built using the MERN (MongoDB, Express.js, React, Node.js) stack, implementing core social media features.

## **Features**

- User authentication (signup, login, logout)
- Post creation and deletion
- Image uploading
- Like and comment functionality
- User profile management
- Follow/unfollow users
- Real-time notifications

## **Prerequisites**

- Node.js (v16+)
- MongoDB
- npm or yarn

## **Technology Stack**

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **State Management**: Redux
- **Authentication**: JWT
- **Image Storage**: Cloudinary
- **Styling**: Tailwind CSS

## **Installation**

1. Clone the repository
   ```bash
   https://github.com/Avanish0508/Instagram-Clone-App.git
   ```

2. Install backend dependencies
   ```bash
   cd instagram-clone/backend
   npm install
   ```

3. Install frontend dependencies
   ```bash
   cd ../frontend
   npm install
   ```

4. Create `.env` files in both backend and frontend directories


## **Running the Application**

1. Start backend server
   ```bash
   cd backend
   npm start
   ```

2. Start frontend development server
   ```bash
   cd frontend
   npm start
   ```

## **Deployment**

- Backend: Heroku, Railway, or DigitalOcean
- Frontend: Netlify, Vercel
- Database: MongoDB Atlas
